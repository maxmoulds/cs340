This is the CS340 Project Website for the Harry Potter Database

Menu of operations

  • List, Add, Update and Delete Characters (perform INSERT, SELECT, UPDATE and DELETE operations on a single entity)
  • Change Which classes a character is enrolled in (change M-to-M relationships)
  • Change House of a Character (change 1-to-M relationships)
  • Show all students from a house
  • List and add new houses
  • List and add new classes
  • List and add new items
  • Search characters by Last name
  • Filter people by Certifications they have
